Welcome to SAPP's documentation!
================================

SAPP is a server extension for Halo PC/CE 
which brings many features to the table including scripting capabilities.
This is the easiest and fastest tool to set up a dedicated server
for Halo PC/CE multiplayer.

.. note:: This documentation was written for SAPP version 10.2 and 
   may not completely apply for
   future or past versions.
